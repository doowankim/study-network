# React Project
## 자기주도적으로 스터디를 만들고 운영하는 웹 서비스

* 웹 프론트엔드(React / Redux / styled-components)
* 백엔드(Node.js / Express)
* 데이터베이스(MongoDB)

- [ ] AWS를 이용한 dev 서버 세팅


#### 페이지 구상

### 주요 기능

- [x] 회원가입
- [x] 로그인
- [x] 마이페이지
- [x] 글쓰기
- [x] 게시판
- [ ] 쪽지(message) 주고 받기

### 폴더 구조

* common : 공통 함수
* component : 재사용 컴포넌트
* reducers : 리듀서
* utils : 토큰 불러오기, 내보내기
* validation : 오류 검사
* config : 토큰 유효성 검사
* model : 데이터베이스
* route : 각 Api 구조 설계
